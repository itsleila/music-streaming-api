# Music Streaming API

Uma API REST desenvolvida com Java, Spring Boot e conceitos de Domain-Driven Design (DDD) para simular uma plataforma de streaming de música semelhante ao Spotify.

- SOLID
- Arquitetura em camadas
- Boas práticas REST
- DTOs para comunicação externa
- JPA/Hibernate para persistência
- H2 Database para desenvolvimento

## Sobre o Projeto

A aplicação permite que usuários realizem operações relacionadas ao consumo de música e gerenciamento de assinaturas, incluindo:

- Cadastro de usuários
- Gerenciamento de músicas
- Criação e gerenciamento de playlists
- Favoritar músicas
- Cadastro de métodos de pagamento
- Assinaturas de planos
- Alteração e cancelamento de assinaturas
- Processamento de transações
- Validações antifraude

## Tecnologias Utilizadas

- Java 21
- Spring Boot
- Spring Data JPA
- Lombok
- H2 Database
